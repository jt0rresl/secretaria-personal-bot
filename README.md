# Secretaria Personal Bot 🤖

Bot de Telegram con IA para organización personal.

## Variables de entorno en Railway

| Variable | Valor |
|---|---|
| `TELEGRAM_TOKEN` | Token de @BotFather |
| `SUPABASE_URL` | URL de tu proyecto Supabase |
| `SUPABASE_KEY` | Anon key de Supabase |
| `ANTHROPIC_KEY` | API key de Anthropic |

## Comandos del bot

- `/start` — Bienvenida y muestra tu chat ID
- `/memoria` — Ver lo que recuerda de ti
- `/reset` — Borrar toda la memoria

## Uso

- Escribe **"buenos días"** cada mañana para recibir tu organización del día
- Cuéntale tareas, proyectos, preocupaciones — él lo recuerda todo
- Pregúntale **"¿qué hago ahora?"** cuando estés bloqueado
